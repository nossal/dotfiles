# Setup
1. ```git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim```

2. ```mv .vimrc ~/``
3. ```$ vim +PluginInstall +qall```

4. ```$ cd ~/.vim/bundle/youcompleteme```

  ```$ ./install.py --clang-completer --tern-completer --racer-completer```
